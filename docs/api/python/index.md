# Python API Reference

Complete reference for the `grafeo` Python package.

## Installation

```bash
uv add grafeo
```

## Quick Start

```python
import grafeo

db = grafeo.GrafeoDB()
with db.session() as session:
    session.execute("INSERT (:Person {name: 'Alice'})")
```
